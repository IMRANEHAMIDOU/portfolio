# Instructions de déploiement

## 1. Créer le repository sur GitHub
- Allez sur https://github.com/new
- Nom : `portfolio`
- Visibilité : **Public** (nécessaire pour GitHub Pages gratuit)
- Ne cochez PAS "Initialize with README"

## 2. Après création, exécutez :
```bash
git push -u origin main
```

## 3. Activer GitHub Pages
1. Allez dans **Settings** du repository
2. Cliquez sur **Pages** dans le menu de gauche
3. Sous **Source**, sélectionnez :
   - Branch : `main`
   - Folder : `/ (root)`
4. Cliquez sur **Save**

## 4. Votre portfolio sera accessible à :
**https://imranehamidou.github.io/portfolio/**

## Note
Si vous voulez que l'URL soit `https://imranehamidou.github.io/` (sans /portfolio), créez un repository nommé exactement `IMRANEHAMIDOU.github.io` au lieu de `portfolio`.
