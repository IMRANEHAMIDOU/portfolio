# Dossier Images

Placez vos images dans ce dossier :

## Photo de profil
- **Nom du fichier** : `profile.jpeg` (ou `profile.jpg`, `profile.png`)
- **Taille recommandée** : 400x400 pixels minimum
- **Format** : JPEG, JPG ou PNG
- **Description** : Votre photo de profil professionnelle

## Images des projets
Nommez vos images de projets comme suit :
- `solani.png` - Système SOLANI
- `laiterie-sahel.png` - LA LAITERIE DU SAHEL
- `ust-niger.png` - Système de Gestion UST-NIGER
- `formation-ifutur.png` - Formation IFutur
- `saraounia.jpg` - Saraounia Shop

**Recommandations** :
- Taille recommandée : 800x600 pixels minimum
- Format : JPG ou PNG
- Poids : Optimisez vos images pour le web (moins de 500KB par image)

## Note
Si une image n'est pas trouvée, un emoji sera affiché à la place automatiquement.
